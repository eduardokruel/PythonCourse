Variables

Variables are one of the fundamentals of almost any (object oriented) programming language. They allow you to store and modify values that are stored in them.

You might notice that what we want to print in the function is encapsulated by double quotes (""). This is to separate strings (strings of characters), that have quotes surrounding, from variable names, that don't. To test this out create a variable using the assignment operator (=), name it whatever you'd like, assign it to a string, then on a different line print the variable.

Example

name = "Eduardo"
print(name)

Keep in mind that variables do not use quotes and strings do!

Numbers

Numbers are another data type (just like strings) that can be assigned to variables. The following symbols can be used in certain math operations:

  • Returns numbers
    • + (Addition)
    • - (Subtraction)
    • * (Multiplication)
    • / (Division)
    • ** (Exponent)
    • % (Modulation)
  • Returns booleans (True or False)
    • < (Less than)
    • > (More than)
    • <= (Less or equal than)
    • >= (More or equal than)
    • == (Equal to)
    • != (Not equal to)

Feel free to experiment with any of these to see the outputs inside a input function (without quotes)

Example

print(5>4)

Functions

Functions are on of the core parts of many programming languages as they allow you to repeat tasks that are common. Functions require a few things:

  • The key word def before the function name.
  • A set of parenthesis with any parameters inside of them
  • A colon after the second parenthesis.
  • Code inside the function should be indented.
  • A return statement with what to return.

Example

def multiplyByTwo(item):
  return item * 2

Pro Tip: Test out this function by using the following statement:

Example

print(multiplyByTwo(4))

The result should be 4!

Conditionals

There are 3 main conditional statements:

  • if
  • elif
  • else

The main reason to use conditional statements is to run certain parts of the program when specific conditons are met. If and elif (combination of else and if) both require a condition after them to determine to run the code or not. If the result of the condition is True it will run, if it is False, it will not. The else statement, if included, will run if no if or elif statement has run before.

Example

if age<=18:
   state="Kid"
elif age>65:
   state="Retired"
else:
   state="Active"

You can see how useful these statements can be when applied correctly to your program!

Libraries

Libraries are one of the reasons that Python is becoming such a dominant language. There are many ways to import libraries in Python, here is the most common way:

Example

import math as math

Let's reread this code, you just imported the library math, and you will reference it as math as well. For example, this math library has many ready-to-go function (complete list down below on the documentation), I will use the variable pi for example.

Example

import math as math
print(math.pi)

From the result we can see that worked as it printed Pi! As we will reference this library as math, every reference to a variable or function of that library will require that we use math. before them as shown above.
